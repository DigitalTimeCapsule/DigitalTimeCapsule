# Project Summary

## Overview of Languages, Frameworks, and Main Libraries Used
The project is built using the following technologies:
- **Backend**: Java with Spring Boot framework.
- **Frontend**: JavaScript with React framework.
- **Build Tool**: Maven for backend dependency management and build processes.
- **Main Libraries**: 
  - Spring Boot (for backend application development)
  - React (for frontend user interface)

## Purpose of the Project
The project, titled "Digital Time Capsule," aims to provide a platform where users can create, store, and manage digital time capsules. Users can interact with the application through a web interface, allowing them to log in and manage their digital content securely.
