# Digital Time Capsule

This project is a digital time capsule allowing users to lock away information (text, videos, photos, and voice memos)
for a chosen period of time before it can be retrieved.

## Prerequisites

- Node.js and npm installed
- Maven installed
- Java 17 or higher installed

## How to Run the Application

1. Open two terminal windows.
2. In the first terminal, run:
   ```bash
   cd frontend
   npm install
   npm start

3) In the second terminal, run:
   ```bash
   cd backend/digitaltimecapsule
   mvn spring-boot:run
